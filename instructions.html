Установка и запуск с нуля

Эта инструкция написана для человека, который никогда раньше не устанавливал программы через командную строку. Просто выполняй шаги по порядку — пропускать ничего не нужно.

01

Создаём папку

Всё, что нам нужно, будет храниться в одной папке на компьютере. Создадим её первой.

1

Открой Рабочий стол

Сверни все окна и посмотри на Рабочий стол компьютера.

2

Создай новую папку

Нажми правой кнопкой мыши на свободном месте Рабочего стола → выбери «Создать» → «Папку».

Назови папку obzvon (можно латиницей, без пробелов).

💡 Запомни, где находится эта папка — в неё мы будем класть все файлы дальше.
02

Устанавливаем Python

Python — это программа, которая умеет выполнять наш скрипт. Без неё ничего не запустится.

1

Перейди на сайт python.org

Открой браузер и введи в адресной строке: python.org/downloads

2

Скачай установщик

Нажми большую жёлтую кнопку «Download Python 3.x.x». Файл скачается в папку «Загрузки».

3

Запусти установку

Открой скачанный файл двойным кликом.

⚠️ Самый важный шаг во всей инструкции: на первом экране установки обязательно поставь галочку «Add Python to PATH» внизу окна. Если её не поставить — придётся переустанавливать Python заново.
Python 3.x.x Setup
Add python.exe to PATH
[ Install Now ]   [ Customize installation ]

После галочки нажми «Install Now» и дождись окончания установки.

4

Проверь, что всё установилось

Нажми одновременно клавиши Win + R, введи cmd и нажми Enter. Откроется чёрное окно — это командная строка, мы будем часто в неё заходить.

Введи команду и нажми Enter:

python --version

Если в ответ появилось что-то вроде Python 3.12.0 — всё получилось.

03

Кладём файлы скрипта

Теперь поместим сам инструмент в папку, которую мы создали в первом разделе.

1

Скачай два файла

Тебе нужны два файла:

ФайлЧто делает
scrape_crm.pyсам инструмент (мозг программы)
START.batкнопка запуска (на неё ты будешь кликать)
⬇ Скачать scrape_crm.py ⬇ Скачать START.bat
2

Перемести оба файла в папку obzvon

Обычно скачанные файлы попадают в папку «Загрузки». Найди их там, выдели оба (зажми Ctrl и кликни на каждый), затем перетащи мышкой в папку obzvon, созданную ранее.

⚠️ Оба файла обязательно должны лежать в одной и той же папке. Если они окажутся в разных местах — программа не запустится.
04

Первый запуск

Сейчас мы запустим программу в первый раз. При первом запуске она сама установит всё необходимое — это нормально, просто подожди.

1

Открой папку obzvon

Зайди в папку, где лежат файлы scrape_crm.py и START.bat.

2

Дважды кликни на START.bat

Откроется чёрное окно. Не закрывай его — оно покажет всё, что происходит.

💡 Если Windows спросит «Этот файл может представлять угрозу» — это стандартное предупреждение для любых .bat файлов. Нажми «Подробнее» → «Выполнить в любом случае».
3

Подожди установку библиотек

В первый раз программа сама скачает и установит всё необходимое. Это может занять 1–2 минуты — просто не трогай окно.

4

Появится меню

Когда установка завершится, увидишь такое меню:

Командная строка
========================================
  Choose an action:
========================================
  1. Export orders from CRM
  2. Merge MTS call report with CRM orders
  3. Exit
========================================
Enter 1, 2 or 3: ▌

Это меню будет появляться каждый раз — отсюда мы выбираем нужное действие просто вводя цифру.

05

Выгрузка заказов из CRM

Это действие забирает все заказы из CRM и готовит файл для МТС.

1

Введи 1 и нажми Enter

В меню введи цифру 1 и нажми Enter.

2

Войди в CRM через Google

Автоматически откроется окно Chrome. Войди в свой Google-аккаунт, как обычно входишь в почту.

3

Дождись загрузки таблицы

Подожди, пока на странице полностью появится таблица с заказами — это занимает несколько секунд.

4

Вернись в чёрное окно и нажми Enter

Программа сама пройдёт по всем разделам, соберёт данные и покажет прогресс.

5

Результат готов

После завершения в папке obzvon появятся файлы:

crm_orders.xlsxвсе заказы
mts_numbers.xlsxсписок номеров для МТС

Также данные автоматически появятся в твоей Google Таблице, на листе «Заказы».

06

Загрузка в МТС и объединение результатов

Теперь загружаем номера в МТС, запускаем обзвон и забираем результат обратно.

1

Открой кабинет МТС VATS

Перейди в раздел «Автоинформирование» → «Задания» и создай новое задание.

2

Загрузи список номеров

В разделе «Список для информирования» нажми «Загрузить список» и выбери файл mts_numbers.xlsx из папки obzvon.

3

Запусти обзвон и подожди завершения

МТС сам позвонит всем клиентам из списка.

4

Скачай отчёт

После завершения обзвона скачай отчёт по заданию в формате CSV.

5

Положи отчёт в папку obzvon

Перемести скачанный файл (например report.csv) в ту же папку, где лежит START.bat.

6

Запусти объединение

Снова открой START.bat (или используй открытое окно), введи 2 и нажми Enter.

Когда попросит имя файла — просто нажми Enter, чтобы использовать report.csv по умолчанию.

7

Готово

Результат автоматически появится в Google Таблице, на листе «Результаты обзвона», привязанный к каждому заказу по номеру телефона.

07

Если что-то пошло не так

Самые частые ситуации и как их решить. Нажми на вопрос, чтобы увидеть ответ.

Чёрное окно закрылось само, я ничего не успел прочитать

Не запускай файл scrape_crm.py напрямую двойным кликом — всегда открывай именно START.bat. Он не закрывается сам и показывает все сообщения до конца.

Появилась ошибка «No such file or directory»

Это значит командная строка открыта не в той папке. Введи команду, заменив путь на свой:

cd C:\obzvon
Ошибка «FileExistsError: lock file exists»

Предыдущий запуск завершился неправильно и оставил файл блокировки. Удали его командой (замени «ИмяПользователя» на своё):

del C:\Users\ИмяПользователя\.wdm\.wdm-lock-chromedriver-win64

Затем запусти START.bat заново.

Номера в Excel выглядят как 3,75E+11

Это происходит только при открытии файла глазами — на саму загрузку в МТС это не влияет. Программа уже создаёт файл mts_numbers.xlsx в правильном формате, проблем при загрузке быть не должно.

МТС не принимает файл со списком номеров

Попробуй по очереди:

— Создать новое задание вместо использования старой копии

— Открыть кабинет МТС в режиме «инкогнито» (Ctrl+Shift+N)

— Убедиться, что выбран правильный файл именно mts_numbers.xlsx
