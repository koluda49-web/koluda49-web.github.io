📋 Что делает этот инструмент

Скрипт автоматически выполняет полный цикл обзвона клиентов:

  1. Выгружает список заказов из CRM (ok-crm.com)
  2. Загружает номера телефонов в МТС VATS и запускает роботный обзвон
  3. Робот звонит клиентам — они нажимают «1» (готов) или «2» (не готов)
  4. Скачивает отчёт МТС и загружает результаты в Google Таблицу
✅Результат: в Google Таблице обновляется лист «Результаты обзвона» — по каждому заказу видно ответ клиента.
🗺️ Выбор маршрута при выгрузке

При запуске пункта 1 (или пункта 5 — полный цикл) скрипт сразу спросит, какой маршрут выгружать — ещё до открытия браузера:

=======================================================
  Выбери маршрут для выгрузки:
=======================================================
  0. Все маршруты
  1. б
  2. а
  3. в
  4. минск
  5. 6м1
  6. 11м
=======================================================
  Введи цифру: _
💡0 — выгрузить все маршруты сразу (как раньше). Любая другая цифра — только выбранный маршрут. Удобно когда нужно запустить обзвон только по одному направлению.
💻 Первичная установка (один раз)
1
Создать папку C:\obzvon

Все рабочие файлы должны лежать в этой папке.

2
Скопировать три файла в папку

START.bat  ·  scrape_crm.py  ·  config.txt (логин/пароль МТС — хранить в тайне)

3
Установить Python (если ещё не стоит)

При первом запуске START.bat откроется браузер со страницей загрузки. При установке обязательно поставь галочку «Add Python to PATH».

4
Запустить START.bat

Дважды кликни по START.bat — откроется чёрное окно консоли. При первом запуске библиотеки установятся автоматически (1–2 минуты).

🔄 Полный цикл — пункт 5 меню

Самый простой способ: выбери пункт 5 — «ПОЛНЫЙ ЦИКЛ: пункты 1+2+3 подряд» и следуй подсказкам на экране.

⏱️Весь цикл занимает около 20–30 минут. 15 минут из них скрипт ждёт завершения обзвона в МТС — обратный отсчёт виден в консоли.
Шаг 1Выгрузка заказов из CRM
  1. В консоли появится меню выбора маршрута — введи цифру: 0 для всех маршрутов, или номер конкретного (б, а, в, минск, 6м1, 11м)
  2. Откроется браузер Chrome с сайтом ok-crm.com
  3. Войди в CRM через Google, если появится страница входа
  4. Когда увидишь таблицу с заказами — нажми Enter в чёрном окне консоли
  5. Скрипт автоматически прокрутит таблицу и соберёт все заказы
  6. Файлы crm_orders.xlsx и mts_numbers.xlsx сохранятся в C:\obzvon
Шаг 2Запуск автообзвона в МТС
  1. Браузер откроет МТС VATS — вход выполнится автоматически
  2. Скрипт создаст задание, введёт текст сообщения и загрузит номера
  3. ПАУЗА: вручную выставить параметры дозвона (см. блок ниже)
  4. ПАУЗА: вручную включить тумблер «Включить информирование» (см. ниже)
  5. Скрипт нажмёт «Сохранить» — обзвон запустится автоматически
Шаг 3Отчёт МТС → Google Таблица
  1. После 15-минутного ожидания браузер откроет список заданий МТС
  2. Скрипт скачает отчёт и сопоставит результаты с заказами по номеру телефона
  3. Данные загрузятся в Google Таблицу (лист «Результаты обзвона») — строки автоматически подсветятся цветом и появится колонка «Открыть в CRM» с прямой ссылкой на каждый заказ
  4. В папке C:\obzvon появится файл crm_with_results.xlsx с теми же ссылками и цветами
✋ Ручные остановки — важно не пропустить

В процессе шага 2 скрипт дважды останавливается и ждёт действий вручную.

📞 Остановка 1 — Параметры дозвона

Найди на странице МТС блок «Параметры дозвона» и заполни два поля:

  • Максимум попыток — выбрать «3» из выпадающего списка
  • Перезванивать через — вписать «5» (минут)

После этого нажми Enter в чёрном окне консоли.

🔘 Остановка 2 — Включить информирование

Найди тумблер «Включить информирование» и кликни по нему — он должен стать синим/зелёным.

Подсказка: когда выключен — рядом написано «Задание нужно будет запустить самостоятельно». Когда включён — «Задание запустится автоматически».

После этого нажми Enter в чёрном окне консоли.

✅ Финальная проверка перед сохранением

Перед нажатием Enter для сохранения задания убедись, что на экране:

  • Правильный текст голосового сообщения
  • Правильное количество номеров
  • Расписание: Пн–Пт, 07:00–21:00
  • Максимум попыток: 3, перезвон через: 5 минут
  • Тумблер «Включить информирование» — синий (включён)
⚠️Если что-то не так — закрой окно браузера вместо Enter. Потом запусти пункт 2 заново.

🔧 Частые проблемы и решения
ПроблемаРешение
Python не найден Скачай с python.org, при установке поставь галочку «Add Python to PATH», перезапусти START.bat
FileNotFoundError: crm_orders.xlsx Сначала запусти пункт 1 — файл ещё не создан
Отчёт МТС неполный (мало записей) Обзвон ещё не завершился. Подожди и запусти пункт 3 снова
Браузер закрылся сам во время работы Перезапусти нужный пункт заново — скрипт создаст новую сессию
МТС не вошло автоматически Войди вручную в открывшемся браузере, потом нажми Enter в консоли
Консоль закрылась сама (ошибка) Сделай скриншот ошибки и передай разработчику
