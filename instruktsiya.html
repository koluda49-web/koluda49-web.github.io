Работа с оптовыми отгрузками

Ежедневные операции — курс, документы, отгрузки
1.1 Курс доллара
  • Устанавливается каждый день
  • По БСБ Банку (наличный, РБ)
Таблица курса — Google Sheets
1.2 Документы на водителя
  • Договор подряда — обязателен
  • Авто не в собственности → договор поручительства
  • Водитель оформлен → выписывается ТТН
  • На каждый заказ: гарантия + паспорт
Не оформлен → ТТН вручную: вес + номенклатура
Нал/безнал (50×50) — уточнять у бухгалтера
1.3 Перемещение между складами
  • Накладная на провоз (согласование с бухгалтерией)
  • Данные — у Логиста
  • Номенклатура, вес, кол-во — у нач. склада либо вкладка номенклатура
1.4 Брак товара
  • Данные → таблица «Опт»
  • ТТН на провоз с расчётом НДС
  • Перемещение → Медвежино Сервис
1.5 Отгрузка на рынки
  • Информация: почта 987 / Коля / PSI
  • ТТН — оформляется вручную
  • Гарантия выдаётся по количеству товара
1.6 Отчёт по ТТН
  • После каждой ТТН → данные в таблицу отчёта
Таблица отчёта по ТТН — Google Sheets

Рассрочка и кредит

Документы, оформление, экземпляры
2.1 Документы
  • Распечатать документы по ссылке в заказе
  • Данные → таблица «Рассрочка Зикмес»
  • Электротранспорт помечать отдельно
  • Договора хранения — вручную: наименование, подпись, печать, номер дилера
Таблица «Рассрочка Зикмес» — Google Sheets
2.2 Особенности
  • Рассрочка фирмы → чек только на сумму первого взноса
  • Все кредиты под отчёт → в бухгалтерию (где бился чек, где нет)
  • В чат каждый день: количество выписанных ТТН
2.3 Печать и экземпляры
  • Рассрочка фирмы → 2 экз.: один в офис, один клиенту (подпись + печать)
  • Кредит → 1 экз., только клиенту
  • К каждому товару стандартно: инструкция + гарантия
3

Загрузка Минск и Межгород

Документы, касса, загрузочный лист, ТТН и ЦРМ
3.1 Подготовка документов
  • Кассовый аппарат: открытие, закрытие, отчёт
  • Маршруты → Документы → Создать для водителя
  • Загрузочный лист + карта с км (Белтрансспутник)
  • В ТТН: модель из номенклатуры, вес и цена — вручную
  • По загрузке: гарантии, паспорта, акты на водителя
3.2 Касса
  • Пт и Пн — касса бьётся на весь электротранспорт
  • Далее через день (если сегодня бита → завтра нет)
  • Чеки ЦРМ и касса Минск — всегда
  • Если у водителя касса → пометка «терминал» в названии маршрута, выдать кассу, отправить в бухгалтерию данные (например: жданко поехал с альфа кассой 16.06)
  • Межгород — кассу не бить (уточнять у бухгалтера)
  • На масло и печи — касса обязательно + фото в бух.
Смотреть по истории сообщений с бухом
Цена масла в ЦРМ → бить её. Нет → бить 10 р.
3.3 ТТН и ЦРМ
  • На каждого водителя → ТТН → в таблицу отчёта (фамилия водителя + дата + номер ТТН)
  • В ЦРМ к каждому маршруту вручную → номер ТТН
  • Всё через @ (собаку) → в ТТН вручную с НДС
  • Убираем собачки и исправляем ошибки вручную
  • ТТН: 1 экз. Остальные — в бухгалтерии
  • Подпись + печать
В номере ТТН стоит единица последней точки — меняем на конечный пункт разгрузки
3.4 Документы по заказам
  • ЦРМ → Загрузки → Создать документы → PDF
  • Гарантия (дата, наим., номер дилера) + паспорт
  • Вкладыш — сертификат СПМ, копия «верна»
  • Электротранспорт и остальное — разные гарантии
  • Замена / возврат / ремонт → акт (в бухгалтерии), акт в ремонт у Юли со штампом шкип
  • К каждому маршруту — чек
3.5 Ежедневный отчёт
  • Малиновской: водитель, дата, ID, наим., рассрочка/кредит/взнос/полная сумма, чек или нет
  • Электротранспорт — до 16:00
  • Возврат велосипеда: ID, наим., причина

Проверка документов и Общие обязанности

Пострейсовый контроль, документооборот, задачи
4 Проверка после рейса
  • Контроль документов и кассы — всё верно
  • Точка не сдана → все документы забираются
  • К каждому заказу: «сдал / не сдал» + нюансы
  • В пометки к маршруту и заказам: «Доки сдал: [имя]» + косяки или всё ок
5.1 Документооборот
  • По месяцам (электротранспорт — отдельно)
  • Все чеки фотографируются → в бухгалтерию
  • В конце месяца — пачка накладных → подписи
  • Возврат / отказ → сообщить в бухгалтерию
5.2 Дополнительные задачи
  • Отчёт по электротранспорту → Малиновской, ежедневно
  • Заказ канцелярии по офису
  • Расширенная гарантия — заполняется вручную
  • Выписать Роме с Медвежки пакет документов
  • Обучение водителей кассе и документам
  • Договора подряда/залога — хранятся отдельно

Сертификаты для электротранспорта

Как выглядят и как подписываются
Акты и сертификаты соответствия — общий вид
Подпись, печать и пометка «Копия верна»
  • Сертификат СПМ — вкладывается к каждому электротранспорту
  • Копия заверяется: подпись + печать + пометка «Копия верна»
  • Оригиналы хранятся в бухгалтерии

Пакет документов для водителя

Полный комплект, который передаётся водителю перед рейсом
В пакет входит:
  • ТТН (товарно-транспортная накладная)
  • Гарантийные талоны на каждый товар
  • Паспорта изделий
  • Загрузочный лист
  • Карта с маршрутом и километражом
  • Акты (замена / возврат / ремонт)
  • Сертификат СПМ (электротранспорт)
  • Кассовые чеки (при наличии)
